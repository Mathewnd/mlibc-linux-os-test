# mlibc-linux-os-test
Minimal mlibc-linux distribution used to run [os-test](https://sortix.org/os-test/) daily.

Results published [here](https://mathewnd.github.io/mlibc-linux-os-test/) after each CI run with latest [Managarm](https://github.com/managarm/managarm) results (taken from their CI) and results from the [os-test results repository](https://gitlab.com/sortix/os-test-results).
